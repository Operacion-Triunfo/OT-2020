Galas de Operación Triunfo 2020

Descripción

En este repositorio se alojan listas CSV con las canciones de las galas y sus artistas originales.

Componentes

  • Archivos CSV de cada gala: contienen los temas e intérpretes originales.
  • format_regexp.js: contiene en formato .js las expresiones de regulares de reemplazo para convertir el csv en un archivo con formato "Autor" - "Canción".
